Jika terdapat 2 value yang sama(dua-duanya bernilai true atau sebaliknya) terdapat operator yang akan menyandingkan mereka agar berbentuk true, yaitu operator AND / &&. Contoh true && true, maka hasilnya akan true.

Atau jika terdapat 2 value yang salah satunya terdapat syarat terpenuhi(antara true atau false) gunakn operator OR / || maka hasilnya akan true.
