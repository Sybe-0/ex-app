Berjalan atau tidaknya True atau False dinamakan sebuah kondisi (conditions). Pernyataan yang membutuhkan kondisi dari true atau false disebut conditionals. Namun juga bisa diseleksi menggunakan pernyataan IF, apakah code tertentu akan terlewati (true) atau tidak (false). Yang tentu saja jika kondisi nya adalah false maka code tersebut akan dilangkahi.
